## Getting the App

<Steps>
  <Step title="Step 1: Install Node.js and Git" icon="folders">
    Freedeck is built on [Node.js](https://nodejs.org/en) and [Git](https://git-scm.com/downloads). You need to download and install them to your computer before using Freedeck.
  </Step>
  <Step title="Step 2: Download the App" icon="copy">
    To actually use Freedeck, you need the App. [You can download the latest version by clicking here.](https://github.com/Freedeck/app/releases/latest/download/Freedeck.exe)
  </Step>
  <Step title="Step 3: Run the App" icon="rectangle-terminal">
    Run the app\! Double click it from your downloads folder or wherever it is.

    <Warning>
      Windows may give you a SmartScreen popup. _This is because Freedeck is not signed, as that costs money._
    </Warning>
  </Step>
</Steps>

Here's full clarification on what certain terms mean.

<CodeGroup>
```text App
The "App" is the native Windows app that allows for system control and auto updates.
```

```text Companion
The "Companion" is the desktop application that opens on your computer that lets you edit the configuration.
```

```text Server
The "Server" is the underlying data processor that handles the tile database
```

```text Client
The "Client" is your phone, or any other connected device that is just a viewer/interactor of the tiles.
```

</CodeGroup>

## Installing Freedeck
Now we'll finally install the Freedeck application in its entirety. From here, names become less ambiguous.

<Steps>
  <Step title="Step 3: Install Freedeck">
    ![installer.png](/images/installer.png)
    From here, you can press Install. 
  </Step>

  <Step title="Step 4: Launch Freedeck">
    Now that you have Freedeck installed, press Launch Freedeck\!
    ![installed.png](/images/installed.png)
  </Step>
</Steps>
